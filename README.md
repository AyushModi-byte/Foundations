# Foundations

Welcome to the **Foundations** repository. This collection of Jupyter Notebooks serves as a comprehensive dive into the core mathematical and algorithmic building blocks of Machine Learning and Data Science. Instead of relying heavily on high-level machine learning libraries, these notebooks focus on implementing foundational algorithms from scratch to build a deep understanding of their inner workings.

## 🚀 Repository Contents

* **[DAY1.IPYNB](https://www.google.com/search?q=DAY1.IPYNB): Custom Matrix Operations**
* Object-oriented implementation of a custom matrix class (`mat`).
* Manual implementation of fundamental matrix operations, including matrix addition, multiplication, and dimensionality handling, laying the groundwork for linear algebra concepts.


* **[DAY2.IPYNB](https://www.google.com/search?q=DAY2.IPYNB): Principal Component Analysis (PCA) from Scratch**
* Step-by-step implementation of PCA applied to the Iris dataset.
* Includes manual computation of mean centering, covariance matrices, determinants, and power iteration for deriving eigenvectors and eigenvalues.


* **[DAY3.ipynb](https://www.google.com/search?q=DAY3.ipynb): Gradient Descent & Linear Regression**
* Object-oriented implementation of the Gradient Descent optimization algorithm (`class GD`).
* Application of Gradient Descent to minimize a standard quadratic function and optimize the Mean Squared Error cost function ($J(w,b)$) for Linear Regression.


* **[DAY4.IPYNB](https://www.google.com/search?q=DAY4.IPYNB): Core Statistics & Probabilistic Models**
* Fundamental statistical functions built from the ground up (mean, variance, standard deviation, z-score normalization, and covariance).
* Implementation of Bootstrapping to compute 95% Confidence Intervals.
* Application of Naive Bayes probability for email spam detection.


* **[DAY5.IPYNB](https://www.google.com/search?q=DAY5.IPYNB): K-Means Clustering**
* Custom K-Means clustering algorithm built using NumPy.
* Features manual Euclidean distance calculations, iterative centroid updates, and cluster assignments evaluated on synthetic blobs.


* **[DAY6.IPYNB](https://www.google.com/search?q=DAY6.IPYNB): Exploratory Data Analysis (EDA)**
* Comprehensive Exploratory Data Analysis performed on the classic Titanic dataset.
* Includes missing value analysis, distribution plotting, survival rate evaluations, and correlation heatmaps.
